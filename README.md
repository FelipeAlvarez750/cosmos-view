# CosmosView

Dashboard interactivo de exploración espacial construido con React y datos reales de NASA.

## Funcionalidades

- **Foto del Día** — Imagen astronómica diaria con descripción oficial de NASA
- **Fotos de Marte** — Galería de imágenes reales tomadas por el rover Curiosity
- **Asteroides** — Objetos cercanos a la Tierra con datos de velocidad y distancia

## Tecnologías

- React + Vite
- Tailwind CSS
- NASA Open APIs
- React Router DOM

## Instalación

```bash
git clone https://github.com/tu-usuario/cosmos-view
cd cosmos-view
npm install
```
